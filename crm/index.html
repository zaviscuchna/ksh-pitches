Vlož data od kolegy (JSON text). Záznamy se sloučí — duplicity se přeskočí.

Jak synchronizovat s týmem?

  1. Každý člen týmu si otevře tuto stránku ve svém prohlížeči.
  2. Data se ukládají lokálně v prohlížeči — každý vidí svoji kopii.
  3. Kdykoli chceš sdílet záznamy, klikni Exportovat → zkopíruj text → pošli kolegovi.
  4. Kolega klikne Importovat → vloží text → záznamy se sloučí bez duplikátů.
  5. Alternativně stáhni JSON soubor a sdílej ho přes disk/email.

💡 Tip: Domluvte se, kdo je „správce" a pravidelně sdílí master export ostatním.
