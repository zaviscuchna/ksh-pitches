Nastavení KSH Outreach Tracker
Jednorázové propojení s databází. Stačí udělat jednou — ostatní v týmu to nepotřebují.
1
Vytvoř free účet na Supabase
Jdi na supabase.com → Start your project → přihlas se přes GitHub. Pak New project (libovolný název, heslo si zapiš).
2
Vytvoř tabulku — zkopíruj SQL níže
V Supabase panelu jdi do SQL Editor → New query → vlož tento kód a klikni Run:
CREATE TABLE contacts ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, company text NOT NULL, url text, status text DEFAULT 'osloveno', method text DEFAULT 'email', who text, date date, notes text, created_at timestamptz DEFAULT now() ); -- Povol čtení/zápis bez přihlášení (pro tým): ALTER TABLE contacts ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_all" ON contacts FOR ALL USING (true) WITH CHECK (true);
3
Zkopíruj přihlašovací údaje
V Supabase: Settings → API → zkopíruj Project URL a anon public klíč níže.
Údaje se uloží jen do tvého prohlížeče.
